Reference: Comprehensive Rust Cross-Map

Where Google's Comprehensive Rust course lines up with each lesson here — and what it covers that this course doesn't (yet).

Comprehensive Rust is a different animal from this course: a 4-day, lecture-style, breadth-first curriculum with runnable example slides and instructor speaker notes, versus this course's narrow, TDD stub-fill, backend-tied approach. Don't read it instead of doing a lesson's exercise — the whole point here is that you write the body yourself against a failing test, and Comprehensive Rust's examples are filled in. Use it after a lesson, as a second angle on the same concept, or to pick up vocabulary/terminology this course doesn't spell out.

What it is

Free course from Google's Android team, covering the full language plus deep dives (Android, Chromium, bare-metal, concurrency, idiomatic Rust, unsafe). See RESOURCES.md for how it fits alongside the Book, Rust By Example, and Rustlings.

Explicitly out of scope here

Present in Comprehensive Rust, excluded by MISSION.md on purpose — not gaps, decisions:

  • Unsafe Rust — this course's backend domain has essentially no unsafe code. Not where the leverage is.
  • Macro authoring — a hand-written parser or DSL grammar is explicitly a "read and admire" kind of file in the concept map, not a "build one yourself" target.
  • Android, Chromium, and bare-metal deep dives — not relevant to this course's domain (backend services, not embedded/mobile/browser code).
