Reference: Rustlings Cross-Map

Where Rustlings lines up with each lesson here — and what it covers that this course doesn't (yet).

Rustlings is closer in spirit to this course than Comprehensive Rust or Rust By Example are: it's a series of small, deliberately broken programs you fix until they compile and pass — the same fix-it-until-the-test-is-green shape as every stub-fill exercise in this course, just without the backend-domain tie-in. Use it for extra reps on one specific mechanic when a lesson's single exercise isn't enough practice, not as a substitute for a lesson's field study.

What it is

Official exercises from the Rust project itself (rust-lang/rustlings on GitHub), organized into numbered topic directories (e.g. 07_structs, 16_lifetimes) plus a quizzes/ directory. Exercise numbers can drift across Rustlings releases, so this page and every lesson's Rustlings link cite exercise directories by name, not position. See RESOURCES.md for how it fits alongside the Book, Comprehensive Rust, and Rust By Example.

What Rustlings covers that isn't a lesson here yet

Same policy as the Comprehensive Rust map: these aren't queued lessons — per MISSION.md, curriculum content is pulled by outcomes, not pre-planned. This is a reading list if you want extra reps before a lesson exists for the topic.

Topic Rustlings exercise set Status
Variable bindings, mut, shadowing (in depth) 01_variables Thin gap — folded a short mention into Lesson 1's Primary source callout rather than given a dedicated lesson; the mechanic is small enough that Lesson 1's own exercise already exercises it in passing.
Testing idioms (#[should_panic], test organization) 17_tests Already on NOTES.md's "not yet written, deliberately" list.
Macro authoring (declarative macros) 21_macros Explicitly out of scope — see below, same reasoning as the Comprehensive Rust map's macro-authoring exclusion.
Quizzes (multi-topic, multiple-choice review) quizzes/ directory Not a lesson-format fit as a broad assessment — these are longer, multiple-choice reviews of several prior topics at once, and this course avoids multiple-choice per the teach skill's own rule against giving away answers through option formatting. That's a different tool from cumulative retrieval, though: this course's own tiny closed-book checkpoints (Lessons 4, 7, 10, 13, 16, 18 — one or two short-answer questions, ~2 minutes, immediate <details> feedback) are cumulative retrieval done at lesson scale instead of quiz scale. Rustlings' quizzes/ directory is still fine to do independently once several lessons are behind you, as a longer optional review — just not a substitute for the in-lesson checkpoints.

Explicitly out of scope here

Present in Rustlings, excluded by MISSION.md on purpose — not gaps, decisions:

  • Macro authoring (21_macros) — a hand-written peg-based DSL grammar is explicitly a "read and admire" kind of file in the concept map, not a "build one yourself" target. Same exclusion as the Comprehensive Rust map.
