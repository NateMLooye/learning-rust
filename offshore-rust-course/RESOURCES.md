# Rust for Offshore Access Backends — Resources

## Knowledge

- [The Rust Programming Language ("The Book")](https://doc.rust-lang.org/book/) — official, free.
  The canonical introduction to Rust. Use for: the first-pass explanation of any concept before a
  lesson touches it — ownership, borrowing, enums, traits, error handling, generics, concurrency.
- [Comprehensive Rust](https://google.github.io/comprehensive-rust/) — Google, free.
  A breadth-first course with runnable example slides. Use for: a second angle on a concept after a
  lesson here. See [`reference/0003-comprehensive-rust-map.html`](./reference/0003-comprehensive-rust-map.html)
  for the lesson-by-lesson cross-map. Its examples are filled-in — read for explanation, not to copy.
- [Rust By Example](https://doc.rust-lang.org/rust-by-example/) — official, free.
  Runnable snippets by concept. Use for: idiomatic syntax after you already understand the concept.
- [Rustlings](https://rustlings.rust-lang.org/) — official Rust project exercises, free.
  Small broken programs you fix until they pass — the closest thing on this list to this course's
  own stub-fill pedagogy. Use for: extra reps on one mechanic. See
  [`reference/0005-rustlings-map.html`](./reference/0005-rustlings-map.html) for the cross-map.
  Exercise numbers can drift across releases — cite exercise directories by name (e.g. `07_structs`).
- [The Cargo Book](https://doc.rust-lang.org/cargo/) — official, free.
  Reference for workspaces, features, profiles, and dependency resolution.
- [rust-analyzer manual](https://rust-analyzer.github.io/manual.html) — official, free.
  Explains what the LSP does (inlay hints, clippy-on-save, macro expansion).
- [Clippy lints index](https://rust-lang.github.io/rust-clippy/master/) — official, free.
  Searchable index of every clippy lint with rationale.
- [Jujutsu (jj) — tutorial and CLI reference](https://jj-vcs.github.io/jj/latest/) — official, free.
  Canonical jj documentation. Use for: every version-control lesson (changes vs. commits,
  working-copy auto-commit, `squash`/`split`, conflicts as first-class state).
- [tokio.rs](https://tokio.rs/) — official, free.
  The async runtime documentation, for when a backend service needs async I/O.
- [serde.rs](https://serde.rs/) — official, free.
  Serialization/deserialization for config and API payloads in a backend service.

## Wisdom (Communities)

- [r/rust](https://reddit.com/r/rust) — large, generally high-signal subreddit for the language.
- [Rust users forum](https://users.rust-lang.org/) — official, well-moderated, strong for
  borrow-checker and language questions.
- [jj (Jujutsu) Discord](https://discord.gg/dkmfj3aGQN) and
  [GitHub Discussions](https://github.com/jj-vcs/jj/discussions) — official jj channels.
